# 초보자용 온보딩 UX

## 목표

처음 쓰는 사용자가 한 번의 흐름 안에서 아래를 끝내야 합니다.

1. 설치
2. Paperclip 실행
3. 재부팅 후에도 계속 살아있게 설정
4. 자연어로 첫 회사 생성

## 초보자 UX 원칙

- 처음에는 Paperclip 내부 용어를 숨긴다
- 기본값은 로컬 + 안전 + self-host
- 설정 파일보다 살아 있는 첫 회사를 먼저 보여준다
- 고급 설정은 나중에 펼쳐 보이게 한다
- 복구는 항상 쉽게: 상태 확인, 재시작, 로그 보기

## 추천 첫 실행 흐름

1. `scripts/one_click_install.sh` 실행
2. 원하면 자동 시작 켜기
3. 자연어 bootstrap dry-run 실행
4. 승인 후 실제 생성
5. 대시보드 열기
6. Hermes에서 이어서 사용
   - `리서치 회사 하나 만들어줘`

## 원클릭 설치가 해야 할 일

- 필수 명령 확인
- Python 의존성 설치
- 필요하면 Paperclip 백그라운드 실행
- health check
- 원하면 자동 시작 등록
- 다음에 바로 실행할 자연어 명령 예시 출력

## 자동 시작 UX 문구

기술 용어 대신 이렇게 말합니다.

- 컴퓨터를 켜면 자동으로 시작
- 백그라운드에서 계속 실행
- 멈추면 자동으로 다시 시작

## 성공 화면에 꼭 보여줄 것

- 대시보드 URL
- 현재 실행 상태
- 자동 시작 상태
- 다음으로 해볼 자연어 명령
